## 3. 堆详解
### 3.1 堆的基本结构
```
1. 堆本身也是一棵二叉树，二叉堆是一棵完全二叉树，
2. 堆中根节点的值是最大(最大堆)的或者最小的(最小堆)
3. 左右节点没有大小关系
4. 子堆也满足上述属性

```
**完全二叉树:** 完全二叉树是缺失了右部分的满二叉树，尽可能的从左至右填满每一层
**满二叉树:** 非叶子节点都有左右节点，且叶子节点都在同一层。
